## Failure Mode

The failure mode is **repair burial**.

Repair burial occurs when a governance architecture contains repair documents, but day-to-day protocol work treats repair as deferred, downstream, exceptional, or optional.

The result is a system that can prove the path was preserved but cannot restore right relationship when the path caused harm or erased affected-party claims.

A perfectly hashed injustice is still injustice.
